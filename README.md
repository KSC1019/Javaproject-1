# 12조 404WeFound
***
## 무엇을 만드나요?

저희는 개발 과정에서 여러 어려움을 겪게 되는 개발자와 학생들을 위해,
 질문과 답변을 공유하고 토론할 수 있는 지식 공유 플랫폼을 만들고자 합니다.

## 왜 "404WeFound" 인가요?

‘404WeFound’는 흔히 볼 수 있는 404 Not Found에서 가운데 Not만 We로 바꾸어서 지었습니다.
